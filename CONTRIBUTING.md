# Contributing to PakWest International

Thank you for your interest in contributing to this project.

## How to contribute

1. Fork the repository on GitHub.
2. Clone your fork locally.
3. Create a new branch for your feature or bug fix.
4. Make changes and test them locally.
5. Commit with a clear message describing the work.
6. Open a pull request against the `main` branch.

## Coding guidelines

- Keep PHP logic in reusable includes and components.
- Keep CSS and JS organized inside `assets/css/` and `assets/js/`.
- Use existing JSON structures in `assets/data/` for content updates.
- Do not commit generated or temporary files from `tmp/`.

## Reporting issues

If you find a bug or want a new feature, open an issue on GitHub with:

- A clear description of the problem
- Steps to reproduce
- Expected behavior
- Any relevant screenshots or logs
